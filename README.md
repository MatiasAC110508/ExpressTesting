# Gardening Store CRUD API

## Description
This is a practice project implementing a RESTful API using Node.js, Express, and MySQL. It performs CRUD (Create, Read, Update, Delete) operations on the `ventas_db` database, which models a gardening store's inventory, customers, employees, and orders.

## Tech Stack
- **Node.js**: JavaScript runtime environment.
- **Express.js**: Web framework for building the REST API.
- **MySQL2**: MySQL client for Node.js, utilizing promises for asynchronous queries.

## Prerequisites
Before running this project, ensure you have the following installed:
- Node.js
- MySQL Server
